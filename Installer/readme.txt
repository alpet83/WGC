Winner Game Cheater. 2.XX Versions
======================================================================
Минимальные требования к системе:
 Компьютер + Windows (98/ME/2000/XP)
 CPU >= 100Mhz (Pentium or AMD K5)
 30Mb FREE RAM (>=DDR333 like).
 ~1Mb на HDD под программу.
======================================================================
Рекомендуемые:
 CPU >= 1000 Mhz 
 RAM >= 128 Mb
 ~>много места под файл подкачки, при использовании поиска
 неизвестного значения
======================================================================
Идеальные:
 CPU >= Athlon64 2800+ 
 RAM >= 512 Mb 
======================================================================
FEATURES:

  Wgc пытается экономить память, в правом нижнем углу показывается количество 
зарезервированой им памяти в килобайтах. Порядка 100 млн. смещений им легко
упаковываются в 1-1.5 метра оперативки.
  
  Wgc  безразлично относится к параметрам поиска: часто распространенные числа
типа BYTE:0, им ищутся также быстро как и мало распространенные вроде
DWORD:12345678.  Конечно в первом случае количество отсевов увеличивается, но
бывает небходимо поискать булевые значения которые равны либо нулю, либо единице.

  В программу встроена справка, которая впрочем обновляется не часто,
поэтому об всех улучениях лучше узнавать из ЭТОГО файла. Более детальную
и удобочитаемую справку лучше искать на сайте. 
======================================================================
INSTALLATION:
============================================================================    
   Программа для корректной работы, требует чтобы
файл chdip.dll располагался в папке
System для Windows 9x/ME или System32 для 
Windows2000/XP.
   Если вы захотите удалить программу
файл chdip.dll придется удалять вручную.
  Автоматическая установка производится пакетными файлами:
   installXP.cmd для Windows XP
   install.bat для Windows 9x

сборка от 8 декабря 2004 (2.50)
 - в программу встроена консоль управления.  
  Эта консоль позволяет осуществлять основные функции WGC прямо в игре, что 
 позволяет не переключатся в программу. 
  Работа с консолью возможна в двух режимах
  1. Режим поиска/отсева. Вызов консоли производится нажатием 
  Ctrl-F11 (поиск) /Ctrl-F12 (отсев)  (или другими назначеными клавишами). 
  В появившемся окне можно ввести значение, тип, и правило отсева. После 
  ввода сразу начнется процесс поиска/отсева. 
  2. Режим просмотра. В этом режиме работают несколько комманд:
  add - добавление найденых значений в таблицу (что на вкладке 3).
  table - просмотр таблицы значений 
  list - просмотр списка чит-кодов
  cheat - перезапись значений (как при нажатии кнопки Cheat)
  set - установка значения, комманда имеет два параметра: номер строки таблицы и 
  нужное значение, например "set 2 1000000", перезаписывает значение находящееся 
  в строке 2.   
  close - закрытие консоли 
 - исправлены несколько ошибок интерфейса.

сборка от 4 мая 2005 (2.65).
  - произведена оптимизация алгоритма поиска значений типа DWORD.   
  - добавлена возможность поиска выровненных значений типа DWORD. 
    В большинстве игр где используются значения типа DWORD (он же long, int), они обычно выравниваются
  по адресам кратным 4. Поэтому теперь WGC по умолчанию ищет с установленным флажком aligned, что позволяет
  сократить количество ложных адресов при поиск и значительно повысить скорость поиска.
 Вот результаты тестов программы с использование поиска значений в процессе на 200 mb:

 [AMD Athlon XP 2000+ 512MiB DDR400] 
   std speed = ~310mb/sec, aligned speed ~= 400mb/sec, 
   std speed spy-mode =~ 465mb/sec, aligned speed spy-mode = 715 mb/sec (prefetch on*)
   
 [Intel Celeron 2400 512MiB DDR333]
   std speed = 376 mb/sec, aligned speed ~= 506mb/sec
   std speed spy-mode =~ 612mb/sec, aligned speed spy-mode = 940mb/sec (prefetch off*)

 *управление флажком prefetch на странице настроек становится возможным при включении spy-mode (HOOK)

сборка от 10 мая 2005 (2.66).
  - были скоректированны ошибки приводившие к возникновению исключений при запуске 
    программы в Win98. Возможно появление аналогичных ошибок если WGC будет запускаться 
    как единственная программа (лучше сначала запустить игру, а в WGC включить автовыбор игры).
  - была немного изменена вкладка с картой памяти - с вкладки поиска перенесен выбор 
    диапазона сканирования памяти и добавлено отображение модулей соответствующих регионам. 
============================================================================
сборка от 5 июня 2005 (2.67).
  - добавлены алгоритмы поиска байтовых и четырех-байтовых (DWORD) значений оптимизированные
    с использованием MMX. Минимальные требования - процессор Intel Pentium III или AMD Athlon XP. Алгоритмы позволяют искать только значения равные образцу. 
  - исправлены ошибки Spy-Mode, теперь включение этого режима проходит без проблем примерно за 1 секунду. Так же отсев в этом режиме теперь выполняется без тормозов.
